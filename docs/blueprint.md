# **App Name**: GuardCard

## Core Features:

- Profile Information Input: Capture the security guard's full name, address, ID number, and card validity date.
- Photo Upload: Allow the user to upload a photo of the security guard for display on the card.
- QR Code Generation: Dynamically generate a QR code based on the security guard's information, for easy scanning and verification. The LLM tool determines relevant properties to include in the QR code, such as photo and time stamp.
- Card Preview: Display a real-time preview of the security guard card as the information is entered.
- Print Functionality: Enable users to print the generated security guard card.

## Style Guidelines:

- Primary color: Navy blue (#1A3A8F) for a trustworthy and secure feel.
- Background color: Light blue (#E0F2F7), a desaturated version of navy, for a clean, professional background.
- Accent color: Gold (#FFD700) to highlight important elements and convey authority.
- Body and headline font: 'PT Sans', a humanist sans-serif that provides a modern, accessible and slightly warm appearance suitable for both headings and body text.
- Use simple, professional icons to represent key information, enhancing clarity.
- Maintain a clean and organized layout, ensuring all information is easily readable and accessible.
- Subtle transition effects when updating the card preview to provide a smooth user experience.
